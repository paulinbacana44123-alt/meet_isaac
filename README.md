# Meet Isaac Ramos

Single-file presentation website for a school oral project.

## How to use
Open `meet_isaac_ramos.html` in your browser.

## For GitHub Pages
1. Create a repository.
2. Upload `meet_isaac_ramos.html`.
3. Rename it to `index.html`.
4. Enable GitHub Pages in repository settings.
5. Use the generated link in your presentation.

## Controls
- Click the buttons to move through topics.
- Use **Left** and **Right** arrow keys to navigate.
